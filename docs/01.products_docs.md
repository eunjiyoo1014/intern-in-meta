# 상품 목록 페이지 프로젝트

## 프로젝트 개요

-주요 기능
신발을 주로 판매하는 홈페이지이다. 모바일에 맞춘 웹페이지를 원한다.

-참여 팀원
유자씨

## 개발 환경

사용한 프레임워크, 라이브러리, 언어 등

예시: React 17.0, JavaScript ES6

## 설치 및 실행 방법

프로젝트를 로컬에서 실행하기 위한 설치 방법과 명령어

npm install
npm run dev

## 컴포넌트 구조

- `ProductList`: 상품 목록을 보여줍니다.
- `ProductCard`: 각 상품의 정보를 보여줍니다.

## 테스트 방법

npm test

## 유의사항 및 알려진 이슈

아직 상품 컴포넌트만 완성되었습니다.
