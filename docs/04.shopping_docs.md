# 쇼핑몰 연동

## 프로젝트 개요

- 프로젝트 목표와 주요 기능에 대한 간략한 설명
  각 페이지 간 연동

- 참여한 팀원
  유자씨

## 개발 환경

React 17.0, JavaScript ES6

## 설치 및 실행 방법

프로젝트를 로컬에서 실행하기 위한 설치 방법과 명령어

npm install
npm run dev

## 프로젝트 구조

ProductPage
CartPage
PaymentPage
CheckoutSuccessPage

## 주요 함수와 기능

BrowserRouter as Router, Routes, Route 기능으로 연동 구현

## 테스트 방법

npm build
npm test
